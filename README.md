# Personal-Financial-Analysis
## Project Overview

This project explores personal financial data from 2023–2024 to uncover patterns in income, expenses, and overall financial health.
The objective is to transform raw transaction data into meaningful insights that support:

Better financial decision-making

Expense optimization

Income diversification strategies.
## Dataset Description

The dataset contains detailed financial transactions with the following fields:

Date:	Transaction date

Type:	Income or Expense

Category:	Classification of transaction

Amount:	Monetary value

Account:	Source or payment method

Spending: Type	Need, Want, or Income

Recurring:	Indicates repeated transactions

Location:	City of transaction
## Tools & Technologies

Data Analysis: Excel / Power BI

Data Visualization: Power BI Dashboard

Documentation: Markdown (GitHub)
